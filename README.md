# SOFT2101 - Lab1
Lab1 Guessing Game
A simple game which lets the player guess a number between 1 and 100, gives a clue how close they are. We are building this for anyone who wants to play a very simple and quick game, and/or anyone who wants to see how a simple java application works.

How to run:
  1. Download zip
  2. Import Java Project your IDE
  3. Run Main
  4. Input your guess into the console
  5. Enjoy!
